# Racing BGM sources

## Selected for gameplay

- Track: `Bouncer`
- Artist: `Of Far Different Nature`
- License: `CC0`
- Source page: <https://opengameart.org/content/bouncer-0>
- Local file: `assets/bgm_bouncer_cc0.mp3`
- Notes: Electro beat, fits arcade racing pacing well. The page description also tags it as suitable for a racing game.

## Good alternatives

- `Liquid Flame` by `Of Far Different Nature`
  - License: `CC0`
  - Source: <https://opengameart.org/content/liquid-flame>
  - Notes: More trance / electro, longer atmosphere, better if you want a smoother highway feel.

- `Hyper Ultra-Racing` by `Bunji Cottontale`
  - License: `CC0`
  - Source: <https://opengameart.org/content/hyper-ultra-racing>
  - Notes: Fast drum and bass, very direct racing-game energy. The source page currently exposes a WAV download.

- `Hyperflight Racing` by `Bunji Cottontale`
  - License: `CC0`
  - Source: <https://opengameart.org/content/hyperflight-racing>
  - Notes: More synth / techno flavor, works well for countdown or time-attack style play.
